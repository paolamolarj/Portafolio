Proyecto

Reconocimiento Facial

Programa en Node.js que toma captura de pantalla del rostro y lo guarda para poder reconocerlo después

Descripción

Objetivo del Proyecto

Crear un sistema de Reconocimiento Facial dónde almacenemos diferentes rostros en una base de datos y nos pueda reconocer en tiempo real y al mismo tiempo.

¿Cómo lo llevamos a cabo?

PROCEDIMIENTO

  • Primero instalamos Python, opencv e imutalls
  • Después creamos carpetas con videos de nuestros rostros y con los códigos que ocupamos.
  • Para poder realizar el sistema ocupamos 3 diferentes códigos: uno para crear la base de datos de los rostros, otro para entrenar los datos y por último ponerlo a prueba.
  • Para el primer código le cambiamos algunos datos y cuando lo ejecutamos se tomaron capturas del video de nuestro rostro.
  • Ya que tenemos las capturas, con el segundo código se clasifican esas capturas por etiquetas y se cambian a color gris.
  • Por último se inserta el tercer código y probamos con los videos de nuestros rostros, ya que hayan reconocido correctamente podemos hacerlo en videostreaming.

RESULTADO

Añadí este proyecto a mi portafolio ya que fue un reto muy grande al empezar mi carrera y fue de los primeros códigos que analicé y pude poner en marcha. No es un proyecto muy grande pero lo considero algo importante dentro del progreso que he tenido en la carrera.
